WORKS

PHP + Bootstrapで製作したECサイト

サイト名：
TaY's Market
https://taysapp.herokuapp.com/

使用技術：
PHP / Bootstrap / Font Awesome

コメント：
初めて自分で制作した作品です。バックエンドはPHPでスクラッチ開発、フロントエンドはBootstrapで整え、アイコンはFont Awesomeを利用しました。

機能説明：
○会員登録機能：名前やメールアドレス等の会員情報を登録。
○ログイン機能：会員登録したメールアドレス、パスワードを用いてログイン。
○ログアウト機能：サイトからログアウトする機能。
○カート機能：商品をカートに追加する機能。個数変更や削除ができる。
○購入機能：カートに入れた商品を購入する機能。ログインしていればお客様情報の確認をして購入、ゲストの場合は名前、住所等のお客様情報を入力してから購入。購入後はカートの中身が空になる。
○メール送信機能（未実装）：購入した商品情報等を登録されたアドレスに送信する機能。不具合により現在停止中。

制作した経緯：
PHPを勉強するにあたり、目標となる制作物をECサイトにしました。理由は普段何気なく利用しているamazonや楽天等のショッピングサイトの、サーバーサイドの動きを自分で作りながら勉強したいを思ったからです。
また、現在PHPではLaravelやCakePHP等のフレームワークを利用すればログイン機能などは簡単に実装できます。ですが私は、サーバーサイドの動きを1から実装し理解を深めたいという考えで、敢えてフレームワークは用いませんでした。

(あくまで「PHPへの理解を深めたい」がメインのため、CSSに関してはBootstrapを用いました)
